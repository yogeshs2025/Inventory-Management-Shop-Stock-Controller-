📦 Shop Stock Controller – Inventory Management System
📌 Project Description

This project is a C++ based Inventory Management System developed using Object-Oriented Programming concepts. It allows users to manage product records efficiently by performing operations such as adding, searching, updating, deleting, and generating reports. The system also ensures data persistence using file handling.

🚀 Features
Add new product
View all products
Search product by ID or name
Update product details
Delete product
Low stock alert report
Total inventory value calculation
File storage using fstream
Input validation and error handling
🛠 Technologies Used
C++
Object-Oriented Programming (OOP)
STL (vector, string)
File Handling (fstream)
▶️ How to Run (Local System)
Step 1: Compile the code
g++ main.cpp -o inventory
Step 2: Run the program
./inventory
🌐 WebAssembly (Optional Deployment)

This project was prepared for WebAssembly deployment using Emscripten.
